ES6 声明变量的六种方法
ES5 只有两种声明变量的方法：var命令和function命令。ES6除了添加let和const命令，另外两种声明变量的方法：import命令和class命令。所以，ES6 一共有6种声明变量的方法。
1.变量声明const和let
块级作用域
说白了只要在{}花括号内的代码块即可以认为 let 和 const 的作用域。
et 和 const 声明只在最靠近的一个块中（花括号内）有效
const 在声明时必须被赋值
2.字符串
第一个用途，基本的字符串格式化。将表达式嵌入字符串中进行拼接。用${}来界定。
const user = ‘world’;
console.log(hello ${user}); // hello world
3.箭头函数
● 不需要 function 关键字来创建函数
● 省略 return 关键字
● 继承当前上下文的 this 关键字
当你的函数有且仅有一个参数的时候，是可以省略掉括号的。当你函数返回有且仅有一个表达式的时候可以省略{} 和 return
